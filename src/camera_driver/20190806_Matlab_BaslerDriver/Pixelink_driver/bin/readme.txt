========================================================
    Matlab Interface for PixeLINK cameras (plFGI)
========================================================

		Directory Contents:	

This is the bin directory of the plFGI. It contains
all mex files, and for every mex file a m file to be
used by the matlab help system.

   The following files are in this directory:

DLL (Mex) files (compiled with PixeLINK SDK 3.1 for Matlab 6):
--------------
plClose.dll	- the plClose function
plDevices.dll	- used internally by the other functions
plGet.dll	- the plGet function
plGrab.dll	- the plGrab function
plIsOpen.dll	- the plIsopen function
plOpen.dll	- the plOpen function
plSet.dll	- the plSet function

M files:
--------------
plClose.m	- helpfile for the plClose function
plDevices.m	- helpfile for plDevices
plGet.m		- helpfile for the plGet function
plGrab.m	- helpfile for the plGrab function
plIsOpen.m	- helpfile for the plIsOpen function
plOpen.m	- helpfile for the plOpen function
plSet.m		- helpfile for the plSet function

Other Files:
--------------
readme.txt	- this file
